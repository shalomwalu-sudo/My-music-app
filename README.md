Musical Fortnite is a beginner-friendly music learning web app designed
to teach music theory and musical instruments step by step.

## Features
- Beginner music lessons (27 lessons planned)
- Interactive piano keyboard
- Quizzes with progress tracking
- Badges and certificates
- Parent / teacher dashboard
- Offline support using localStorage
- GitHub Pages & APK compatible

## How to Run
Open `index.html` in any browser or enable GitHub Pages.

## APK Conversion
Upload this project to AppsGeyser as a Web App ZIP.
